Apa itu HTML?

HTML atau Hypertext Markup Language merupakan sebuah dokumen yang digunakan untuk sebuah konten atau struktur suatu website

Tag

HTML Tag adalah sebuah elemen HTML. HTML Tag pada umumnya memiliki tiga bagian, yaitu pembuka Tag, isi dari suatu Tag, dan penutup Tag. contoh HTML Tag:


		<h1>Tag HTML</h1>
		
Tag HTML

HTML Tag juga biasanya memiliki beberapa atribut, atribut yang biasanya kita temukan adalah 'class'

atribut biasanya memiliki sebuah value atau sebuah isi, dan beberapa atribut juga biasanya akan bekerja tanpa value, contohnya 'disabled' dan 'selected'. contoh HTML Tag atribut:


		<input disabled />
